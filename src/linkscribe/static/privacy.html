Privacy policy

LinkScribe processes media links solely to create a transcript requested by an authenticated user.

Data processed

The service stores the submitted URL, processing status, basic media metadata, and transcript. API tokens are verified and are never included in job records.

Retention

Downloaded media and converted audio are deleted immediately after each job. Job records and transcripts expire automatically after the configured retention period, which defaults to 24 hours.

Sharing

LinkScribe does not sell data. The selected media platform receives ordinary download requests, and the transcript is returned only to authenticated API clients.

Contact

Open a privacy or security issue through the project repository without including private media links or credentials.
